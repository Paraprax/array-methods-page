JavaScript Array Methods Cheatsheet

.forEach()
Use: For transforming each element in a targeted array
Mutative: Yes
Parameter(s): An anonymous function
Action: Mutatively performs the code in the function block on each element in the original array
Returns: undefined

.map()
Use: For creating a new array of transformed copies of each element in a targeted array
Mutative: No
Parameter(s): An anonymous function
Action: Populates a new array with the results of performing the function on each element of the original array.
Returns: The new array

.filter()
Use: For copying only the desired elements from a target array into a new array
Mutative: No
Parameter(s): An anonymous function
Action: Populates a new array with items from the original array that satisfy a given conditional.
Returns: The new array
